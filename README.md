# Velora Fashion Website 
here's the link of site- https://valorafashionnnn.netlify.app/contact

## Project Overview

Velora Fashion is a modern responsive fashion e-commerce website created using HTML, CSS, and JavaScript.
The website includes multiple pages such as:

* Home Page
* Shop Page
* Cart Page
* Contact Page

The project focuses on clean UI design, responsive layouts, product showcasing, and simple cart interaction features.

---

# Features

## Home Page

* Fashion hero section
* Stylish navigation bar
* Featured categories
* Trending products
* Responsive design

## Shop Page

* Product cards with images
* Add to Cart buttons
* Product quantity options
* Fashion accessories, bags, jackets, shoes, etc.

## Cart Page

* Product quantity increase/decrease
* Remove product feature
* Automatic total bill calculation
* Discount and sale calculations
* Responsive cart layout

## Contact Page

* Contact information
* Email and phone details
* Social media icons
* Contact form UI

# Setup Instructions

## Step 1 вҖ“ Download Project

Download or copy all project files into one folder.

---

## Step 2 вҖ“ Open in VS Code

1. Open Visual Studio Code
2. Click:
   File вҶ’ Open Folder
3. Select the project folder

---

## Step 3 вҖ“ Install Live Server

1. Open Extensions in VS Code
2. Search:
   Live Server
3. Install the extension

---

## Step 4 вҖ“ Run Website

1. Open `index.html`
2. Right click anywhere
3. Click:
   Open with Live Server

The website will open in your browser.
